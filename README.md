# Estudos em Python

Este repositório contém exercícios e pequenos programas desenvolvidos em **Python**, com foco em lógica de programação, estruturas condicionais e entrada/saída de dados.

## Objetivo
Praticar conceitos fundamentais de programação, como:
- Entrada de dados pelo usuário
- Condições (`if`, `elif`, `else`)
- Comparações lógicas
- Estruturação de algoritmos simples

## Conteúdos abordados
- Comparação de números
- Verificação de idade
- Condições simples e compostas
- Exercícios introdutórios de lógica

## Tecnologias utilizadas
- Python 3

## Como executar
1. Certifique-se de ter o Python instalado.
2. Execute os arquivos pelo terminal:
```bash
